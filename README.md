# jeera-lite
Jira/kanban style board clone


## Core Features (MVP)

### Authentication

Email/password signup/login
JWT token handling
Protected routes

### Board Management

Create/edit boards
Drag-and-drop interface
Multiple status columns

### Task Operations

CRUD operations for tasks
Title, description, priority
Assignment to users
Due dates
Status updates

### User Experience

Responsive design
Real-time updates
Error handling
Loading states
